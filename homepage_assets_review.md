작성일: 2026-06-08

기준 문서: `homepage_plan.md`

대상 폴더: `homepage_assets`

## 1. 전체 방향

이미지 선정 기준은 `Quiet Korean Studio`입니다.

핵심 기준:

- 전통 한국 분위기
- 작가 프로필 허브에 어울리는 차분함
- 전각이 예술과 글로벌 문화체험으로 확장되는 흐름
- 인사동 작업실의 따뜻한 현장감
- 과도한 관광 상품 느낌을 피하는 절제감

## 2. 추천 이미지 역할

### Hero

역할:

- 첫 화면에서 작품성과 작가성을 바로 전달한다.
- 창조, 부활처럼 대비가 강하고 전각 작품의 힘이 잘 보이는 이미지를 우선한다.

사용 기준:

- 얼굴 초상권 이슈가 없어야 한다.
- 작품 전체가 잘 보이고 크롭해도 의미가 유지되어야 한다.
- 한지 아이보리 배경과 먹색 텍스트에 잘 붙어야 한다.

### Profile / Works

역할:

- 김현숙 작가의 프로필, 작품 세계, 전시 이력을 보조한다.
- 작품과 전시장 컷은 작가 프로필의 신뢰도를 높이는 근거 이미지로 사용한다.

추천:

- 최신 프로필 사진
- 전각 작품 클로즈업
- 뉴욕 전시 관련 작품 및 전시장 컷
- 초기 국내 전시를 보여주는 조용한 갤러리 이미지

### Global Events

역할:

- 국내 기반에서 글로벌 문화 프로그램으로 확장된 흐름을 보여준다.
- 외국인 방문객, 해외 바이어, 여행/MICE 관계자에게 신뢰를 준다.

추천:

- 손님 얼굴이 크게 드러나지 않는 작업 장면
- 외국인 참여가 자연스럽게 보이는 손동작 또는 테이블 컷
- SITM, Connections Luxury Seoul 등 글로벌 행사 맥락이 보이는 자료

주의:

- 얼굴이 선명한 단체 사진은 공개 사용 허가를 확인한다.
- 행사 로고나 배너가 크게 보이는 이미지는 보조 이미지로만 사용한다.

### Experience

역할:

- 전각 체험과 한글 캘리그라피 굿즈를 쉽고 아름답게 설명한다.

추천:

- 도장, 책갈피, 전각 결과물
- 한글 캘리그라피가 적용된 엽서, 파우치, 텀블러, 에코백
- 휴대용 손선풍기와 미니 장식 조명은 결과물이 잘 보이는 컷을 사용한다.
- 작업 과정은 손과 도구 중심으로 사용한다.

## 3. 공개 사용 체크리스트

- 얼굴이 보이는 사진 사용 허가 확인
- 미성년자 사진 보호자 또는 기관 허가 확인
- 행사 로고, 배너, 협력사 표기 노출 가능 여부 확인
- PPT에서 추출한 이미지의 원본 사용 권한 확인
- Dropbox 원본 이미지의 최종 고해상도 파일 여부 확인
- 최종 배포용 이미지는 웹 최적화 후 300KB~1MB대 권장

## 4. 현재 사이트 반영 방향

반영된 방향:

- Hero는 전각 작품 중심으로 구성
- About에는 김현숙 작가 사진과 상세 약력 연결
- Experience는 전각 체험, 한글 캘리그라피 굿즈, 단체 워크숍 카드로 구성
- Global은 해외 행사와 글로벌 방문객 대상 프로그램 흐름을 강조

추가로 개선하면 좋은 항목:

- 손님 얼굴 노출이 적은 글로벌 체험 이미지 추가 확보
- 작업실에서 전각 작품을 들고 있는 최신 작가 사진 촬영
- 작품 하단이 잘리지 않는 정면 이미지 확보
- 체험관 로고와 연구소 명칭을 적절히 보여주는 보조 이미지 확보
